Pengaturan Integrasi

Aplikasi ini sudah terintegrasi secara otomatis dengan sistem AI. Anda hanya perlu mengatur database spreadsheet jika ingin menyimpan data secara online.

Koneksi Spreadsheet (Bank Soal)

Opsional: Untuk menyimpan data permanen ke Google Sheets

Cara mendapatkan URL:
  1. Buat Google Spreadsheet baru.
  2. Pilih menu Ekstensi > Apps Script.
  3. Buat fungsi `doPost(e)` untuk menerima JSON.
  4. Klik Terapkan > Deployment Baru (Pilih jenis Web app, akses: Siapa saja).
  5. Copy URL yang dihasilkan ke kolom di atas.
